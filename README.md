# 🏨 Hotel IQ — AI Revenue Management

Hotel IQ is an AI-powered revenue management dashboard for hotels.

## Features
- 📊 Occupancy & RevPAR dashboards
- 💰 Dynamic pricing recommendations
- 📈 14-day demand forecasting
- 🏁 Competitor rate tracking
- 🤖 AI Revenue Analyst chatbot (powered by Claude)

## Built With
- React
- Recharts
- Claude AI (Anthropic)
